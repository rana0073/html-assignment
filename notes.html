5 HTML TAGS

  1. <div>

    A generic container element that groups content for styling and applying structure to a document.

  2. <a>

    Defines a hyperlink, linking to another document or resource.

  3. <img>

    Embeds an image into an HTML document.

  4. <p>

    Defines a paragraph, used for text content within a document.

  5. <ul>

    Defines an unordered list, typically rendered as a bulleted list of items.
